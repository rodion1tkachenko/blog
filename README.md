# 📖 О проекте

Данный проект представляет собой full-stack блог-платформу, разработанную в рамках изучения дисциплины «Веб-разработка».

Приложение позволяет:

- 📝 Создавать посты
- ❌ Удалять посты
- 🔐 Выполнять авторизацию через JWT
- 👨‍💼 Использовать административную панель
- 💬 Отправлять feedback-сообщения автору
- 👤 Просматривать публичное био автора
- ✏️ Редактировать профиль автора через админ-панель

Проект реализован по клиент-серверной архитектуре:

- ⚛️ Frontend — React
- ☕ Backend — Spring Boot
- 🗄️ База данных — PostgreSQL
- 🔄 Миграции базы данных — Liquibase

# ⚙️ Основной функционал

## 📝 Работа с постами

Администратор может:

- Создавать посты
- Удалять посты

Обычные пользователи могут:

- Просматривать посты

Посты автоматически сортируются по дате создания.

---

## 👤 Профиль автора

На главной странице отображается публичный профиль автора.

Профиль содержит:

- ФИО
- Университет
- Информацию о себе
- Telegram для связи

Редактировать профиль может только администратор.

---

# 🔐 Авторизация

Авторизация реализована через JWT-токены.

После успешного входа:

- Токен сохраняется в localStorage
- Защищённые endpoint требуют Authorization header
- Админские endpoint доступны только пользователю с ролью ADMIN
